# Desafio Power BI - Analisando Dados Financeiros

Este repositório contém a entrega do desafio de projeto da DIO. O foco foi a criação de um relatório detalhado utilizando a base de dados 'Financials'.

## 📂 Arquivos do Projeto
Como este é um repositório focado na clareza, os arquivos principais encontram-se diretamente na raiz:

* **[sample_financials.pbix](./sample_financials.pbix):** Arquivo fonte do Power BI contendo as três páginas do relatório (Produtos, Países e a nova página de Análise de Lucro).
* **[sample_financials.pdf](./sample_financials.pdf):** Versão exportada em PDF para visualização imediata dos gráficos e indicadores desenvolvidos.

## 📊 Estrutura do Relatório
O relatório foi dividido em:
1.  **Vendas por Produto e Segmento:** Visão geral de faturamento.
2.  **Vendas por Países e Lucro:** Foco em performance geográfica.
3.  **Análise de Margem e Mapas (Página 3):** Visuais de mapa para soma de vendas e unidades por país, além de lucro por segmento.
